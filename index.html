Sony Professional Solutions Europe

Software Engineer Intern

Basingstoke, England

July 2021 - July 2022

With the goal of helping Sony personnel make better, data driven decisions with their TV software packages. We developed an internal Python automated testing tool coupled with a web application to view and analyse the results.

Project Goals

  • Automated traversal of Sony's TV software utilising object detection and neural networks
  • Collect adequate test results during traversal
  • Web application for analytics and report generation from result data

Web Application

We needed a platform that would utilise the raw data from our Python application and extract as many useful insights as possible to a non-technical personnel.

I was responsible for picking the stack and developing the application from the ground up, By the end of the placement, we were using it to generate reports that combined data from multiple tests which each ran on their own hardware; providing insights that weren't possible before

Python tool

This tool needed to perform a series of checks while navigating to defined areas of Sony's TV software.

We used the producer-consumer model which suited our multiprocessing design of the software. This gave us enough performance to process each frame in a queue without progressively falling behind the real time frames of the TV software.

We are also able to successfully build an object detection algorithm that'll navigate to areas of the software from any start point.
